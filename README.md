# Zenmap 中文汉化与操作指南

## 项目简介
Zenmap 是 Nmap 的官方图形界面（GUI），用于网络扫描和安全审计。该文档旨在帮助用户：

- 将 Zenmap 界面汉化为中文

## 安装 Zenmap

### Windows
1. 访问 [Nmap 官方网站](https://nmap.org/download.html) 下载 Windows 安装包。
2. 运行安装程序，按提示完成安装。
3. 安装过程中确保勾选 **Zenmap GUI** 选项。

## 中文汉化步骤

1. 打开 Nmap > zenmap > lib > python3.xx > site-packages > zenmapGUI
2. 将下载好的zip文件解压缩替换到目录 
3. 关闭并重新打开 Zenmap，即可显示中文界面。
